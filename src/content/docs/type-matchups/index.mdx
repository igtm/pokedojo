## このページで覚えること

- タイプ相性は 0倍、1/4倍、1/2倍、等倍、2倍、4倍で見る
- 2タイプ持ちは倍率を掛け算する
- 攻めと守りの相性は別物

## 倍率で考える

タイプ相性は「効果はばつぐん」「いまひとつ」だけでなく、倍率で覚えると判断しやすくなります。

例として、こおり技はドラゴンに2倍、ひこうにも2倍です。相手がドラゴン/ひこうなら、2倍と2倍を掛けて4倍になります。

一方で、ほのお技をみず/いわに使うと、みずで1/2倍、いわで1/2倍なので、合計で1/4倍になります。

## 攻めと守りは分ける

みずタイプはほのおタイプに攻撃すると強いですが、でんき技を受ける側になると弱点を突かれます。

「このポケモンは何タイプに強いか」と「何タイプを受けられるか」は分けて考えます。
