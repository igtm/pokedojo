## このページで覚えること

- H/A/B/C/D/S は、能力を短く書くための略称
- HD、HC、HS は、努力値をどの能力へ多く振るかを表すことが多い
- S130族は「素早さの種族値が130のグループ」という非公式な読み方
- 初心者は、細かい調整より「何をしたい配分か」を先に読む

## まず能力の略称

対戦記事では、能力を1文字で書くことがよくあります。
最初は次の表だけ覚えれば十分です。

| 略称 | 能力 | 読み方 |
| --- | --- | --- |
| H | HP | 体力 |
| A | こうげき | 物理技の火力 |
| B | ぼうぎょ | 物理技への硬さ |
| C | とくこう | 特殊技の火力 |
| D | とくぼう | 特殊技への硬さ |
| S | すばやさ | 行動順に関わる速さ |

たとえば「Sが高い」は「すばやさが高い」、「Dが低い」は「とくぼうが低い」という意味です。
HABCDS は公式用語というより、対戦記事でよく使われる略記として覚えます。

## 実数値は何で決まるか

バトルで実際に使われる能力値を、対戦記事では「実数値」と呼ぶことがあります。
実数値は、ポケモンごとの得意不得意、育成、性格などが合わさって決まります。

<MermaidDiagram title="能力値を決める材料" caption="最初は、種族値が土台、努力値と性格で役割に寄せる、と読む。">
{`
flowchart LR
  A[種族値] --> E[実数値]
  B[努力値] --> E
  C[性格補正] --> E
  D[個体値] --> E
  E --> F[火力、耐久、行動順]
`}
</MermaidDiagram>

| 言葉 | 意味 | 初心者の見方 |
| --- | --- | --- |
| 実数値 | バトルで使われる最終的な能力値 | ダメージや行動順に関わる数字 |
| 種族値 | ポケモンごとに決まった基礎能力 | そのポケモンの得意不得意 |
| 努力値 | 育成で伸ばす能力 | 役割に合わせて足す部分 |
| 性格補正 | 能力を上げたり下げたりする補正 | 最速、準速、火力補正に関わる |
| 個体値 | 個体ごとの基礎差 | まずは高い前提で読む。最遅では低くすることがある |

初心者は、いきなり実数値を暗記しなくて大丈夫です。
まず「このポケモンは元から速いのか」「育成でさらに速くしているのか」「性格でも速さを上げているのか」を分けて読みます。

## 努力値振りとは

努力値振りは、育成でどの能力を伸ばすかを決めることです。
ゲーム内では「基礎ポイント」と呼ばれることがあります。
対戦記事では、細かい数字より先に「何に多く振っているか」を短く書くことがあります。

多くの本編作品の対戦では、1つの能力に大きく振る時は252、2つに大きく振って残りを4だけ別の能力へ回す形をよく見ます。
たとえば `CS252 H4` なら、CとSを大きく伸ばし、余りをHに回すという読み方です。
別の作品やルールで育成仕様が変わっても、「何を伸ばして、何を諦めているか」を読む考え方は同じです。

| 表記 | ざっくりした意味 | 何をしたいか |
| --- | --- | --- |
| HA | HPとこうげきに多く振る | 物理で殴りながら耐える |
| HB | HPとぼうぎょに多く振る | 物理攻撃を受ける |
| HC | HPととくこうに多く振る | 特殊で殴りながら耐える |
| HD | HPととくぼうに多く振る | 特殊攻撃を受ける |
| AS | こうげきとすばやさに多く振る | 物理で上から攻撃する |
| CS | とくこうとすばやさに多く振る | 特殊で上から攻撃する |
| HS | HPとすばやさに多く振る | 先に補助技を使い、最低限耐える |

この表記は「その2つだけに必ず252ずつ」という意味とは限りません。
記事によっては、中心になる能力を短く言っているだけの場合もあります。

## よく見る数字の形

努力値配分は、次のように書かれることがあります。

| 表記 | 読み方 | 初心者の見方 |
| --- | --- | --- |
| H252 A252 S4 | HとAに大きく振り、余りをS | HA寄りの物理アタッカー |
| C252 S252 H4 | CとSに大きく振り、余りをH | CSの特殊アタッカー |
| H252 B252 D4 | HとBに大きく振り、余りをD | HBの物理受け |
| H252 D252 B4 | HとDに大きく振り、余りをB | HDの特殊受け |
| H244 B252 D12 | H、B、Dを細かく分ける | 何かを耐えるための調整かもしれない |

初心者は、数字を全部覚えるより、まず大きい数字がどこにあるかを見ます。
`252` がAとSにあるなら攻撃と素早さ、HとDにあるなら特殊耐久を重視していると読めます。

## HD、HC、HSの読み方

ユーザーが見かけやすい `HD`、`HC`、`HS` は、役割がかなり違います。

| 表記 | よくある役割 | 例として見ること |
| --- | --- | --- |
| HD | 特殊受け、特殊技を耐える役 | 相手の特殊アタッカーに後出ししたい |
| HC | 耐久を残した特殊アタッカー | 遅くても1発耐えて強い特殊技を撃ちたい |
| HS | 速めの補助役、先に動きたい耐久役 | 先にちょうはつ、みがわり、回復、設置をしたい |

### HD

HDは、HPととくぼうを伸ばす考え方です。
特殊技を受けたいポケモンでよく見ます。

| 強い場面 | 理由 |
| --- | --- |
| 相手の特殊アタッカーに後出ししたい | とくぼうを高くして受けやすくする |
| 回復技や状態異常で長く戦いたい | HPとDがあるほど場に残りやすい |
| ひかりのかべや持ち物と合わせる | 特殊方面をさらに硬くできる |

### HC

HCは、HPととくこうを伸ばす考え方です。
速さよりも、耐えて強い特殊技を撃つことを重視します。

| 強い場面 | 理由 |
| --- | --- |
| 素早さで勝つより、耐えて反撃したい | HPを伸ばすと行動しやすい |
| 高火力の特殊技を押したい | Cに振ると特殊火力が上がる |
| トリックルームや交代から出す | Sに振らなくても動かしやすい |

### HS

HSは、HPとすばやさを伸ばす考え方です。
火力よりも、先に補助技を使うことや、相手より先に回復することを狙います。

| 強い場面 | 理由 |
| --- | --- |
| 先にちょうはつを入れたい | 相手の回復、設置、積みを止めやすい |
| 先にみがわりや回復を使いたい | 行動順で勝てると安定しやすい |
| 耐久も少しほしい | SだけでなくHにも振る |

## ぶっぱ、無振り、調整

努力値の話では、次の言葉もよく出ます。

| 言葉 | 初心者向けの意味 | 読み方 |
| --- | --- | --- |
| ぶっぱ | ある能力へ最大近くまで振ること | ASぶっぱならAとSを大きく伸ばす |
| 極振り | ぶっぱに近い意味。ある能力へ最大近くまで振る | C極振りなら特殊火力重視 |
| 無振り | その能力に努力値を振っていないこと | 無振りSならすばやさに振っていない |
| 余り | 主要な能力に振った後の残り | 余り4を別の能力へ回す |
| 調整 | 目的に合わせて細かく振ること | ある攻撃を耐える、特定の相手を抜く |
| 耐久振り | H、B、Dに多く振ること | 受けたい攻撃に合わせる |
| 火力振り | AやCに多く振ること | 倒したい相手に合わせる |
| S振り | Sに多く振ること | 抜きたい相手に合わせる |
| S調整 | 特定の相手より速くなるようにSを決める | S130族抜き、準速〇〇抜き |
| Sライン | 素早さの基準や境目 | どこまで抜くかの目安 |

初心者は、最初から細かい調整を作る必要はありません。
まずは、AS、CS、HB、HD、HCのような分かりやすい配分から始めると読みやすいです。

## 最速、準速、無補正

素早さの話では、性格補正も一緒に出てきます。

| 言葉 | 意味 | 見方 |
| --- | --- | --- |
| 最速 | Sに多く振り、Sが上がる性格にする | とにかく上から動きたい |
| 準速 | Sに多く振るが、Sが上がる性格ではない | 火力や耐久を性格で伸ばしたい |
| 無補正 | その能力に性格補正がかかっていない | 速さも火力も中間 |
| 最遅 | Sを下げる性格にし、Sを低くする | トリックルームで先に動きたい |

たとえば、同じポケモンでも「最速」と「準速」では素早さが変わります。
最速は速さ重視、準速は速さに振りながら別の能力も重視する考え方です。
最遅は、性格だけでなく個体値や努力値も低くして、できるだけ遅くする意味で使われることがあります。
初心者は「トリックルームなど、遅い方が得をする時の言葉」と覚えれば十分です。

## S130族とは

`S130族` は「すばやさの種族値が130のポケモンたち」という非公式な言い方です。
種族値は、ポケモンごとに決まっている基礎能力のようなものです。
この「族」は、同じくらいの種族値を持つポケモンをまとめて呼ぶ非公式な言い方です。

| 表記 | 意味 | 初心者の読み方 |
| --- | --- | --- |
| S130族 | 素早さ種族値が130のグループ | とても速い基準 |
| S100族 | 素早さ種族値が100のグループ | 速いかどうかの目安になりやすい |
| S80族 | 素早さ種族値が80のグループ | 中速くらいとして見られやすい |
| A120族 | こうげき種族値が120のグループ | 物理火力が高い目安 |
| C135族 | とくこう種族値が135のグループ | 特殊火力が高い目安 |

「S130族抜き」と書いてあれば、指定された条件のS130族より先に動けるように素早さを調整している、という意味です。
「準速〇〇抜き」「最速〇〇抜き」も同じで、特定の相手より先に動くためのS調整です。
「準速〇〇抜き」なら準速の相手、「最速〇〇抜き」なら最速の相手を基準にしている、と読みます。
ただし、実際の行動順は努力値、性格、ランク補正、持ち物、まひ、天候、特性、トリックルーム、優先度などでも変わります。
`S130族` は「元の速さの目安」であり、試合中に必ず先に動く保証ではありません。

## 記事を読む時の順番

努力値や種族値の表記を見たら、次の順で読みます。

<MermaidDiagram title="努力値表記を読む流れ" caption="数字を暗記する前に、役割へ置き換える。">
{`
flowchart LR
  A[略称を見る] --> B[何を伸ばしているか読む]
  B --> C[役割に置き換える]
  C --> D[誰に勝ちたいか考える]
  D --> E[必要なら細かい数字を見る]
`}
</MermaidDiagram>

| 表記 | まず読むこと | 役割への置き換え |
| --- | --- | --- |
| AS252 | AとSを大きく伸ばす | 物理で上から攻めたい |
| CS252 | CとSを大きく伸ばす | 特殊で上から攻めたい |
| HB252 | HとBを大きく伸ばす | 物理を受けたい |
| HD252 | HとDを大きく伸ばす | 特殊を受けたい |
| H252 C252 | HPを確保して特殊火力を出す | 耐えて特殊技で返したい |
| S調整 | 特定の相手を抜くためにSへ振る | 先に動きたい相手がいる |

## よくある読み間違い

| 読み間違い | 正しい見方 |
| --- | --- |
| HDは必ず最強の耐久 | 物理には弱いことがある |
| HCは遅いから弱い | 耐えて高火力を返す役なら強い |
| HSは火力が低いから無意味 | 先に補助技を使う役なら強い |
| S130族なら必ず先に動ける | スカーフ、能力変化、先制技、トリックルームで変わる |
| ぶっぱ以外は全部上級者向け | 目的が分かれば簡単な調整から読める |

## 簡単な例

ゲンガーを特殊アタッカーとして使うなら、CSに大きく振る考え方があります。
これは「とくこうで火力を出し、すばやさで先に動きたい」という意味です。

一方で、特殊攻撃を受ける役ならHD、特殊火力を出しながら耐える役ならHCという読み方になります。
同じポケモンでも、何をしたいかで努力値の意味は変わります。
