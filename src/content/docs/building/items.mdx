## このページで覚えること

- 持ち物は、1体に1つだけ持たせる追加の役割
- 強い持ち物には、条件やデメリットがある
- 相手の技によって、持ち物を持っていること自体が弱点になることがある
- 初心者は「何を1回だけ保証したいか」から選ぶ

## 持ち物は役割をはっきりさせる

持ち物は、ポケモンの役割を強くするための道具です。
同じポケモンでも、持ち物が変わると動かし方が変わります。

| 持ち物で伸ばすもの | 例 | 試合中の意味 |
| --- | --- | --- |
| 火力 | いのちのたま、こだわりハチマキ、こだわりメガネ | 倒せる範囲を広げる |
| 素早さ | こだわりスカーフ、ブーストエナジー | 先に動く範囲を広げる |
| 行動保証 | きあいのタスキ、オボンのみ | 1回動きやすくする |
| 継続力 | たべのこし、くろいヘドロ | 長く場に残る |
| 受けへの負担 | ゴツゴツメット、たべのこし | 受けながら相手を削る |
| 展開補助 | ひかりのねんど、メンタルハーブ | 壁、トリックルーム、補助技を通しやすくする |

持ち物を選ぶ時は、最初に「このポケモンに何をしてほしいか」を決めます。
火力を出したいのか、先に動きたいのか、1回だけ必ず動きたいのかで選ぶ持ち物は変わります。

## 初心者が最初に覚える持ち物

まずは、次の表だけで十分です。
細かい専用アイテムより、役割とデメリットをセットで覚えます。

| 持ち物 | 何が強いか | 主なデメリット |
| --- | --- | --- |
| きあいのタスキ | HP満タンなら強い攻撃を耐えて1回動きやすい | ステルスロック、まきびし、天候、先制技の連続などで崩れる |
| いのちのたま | 技を選び直せる火力強化 | 攻撃するたびにHPが減る |
| こだわりハチマキ | 物理火力が大きく上がる | 同じ技しか選べなくなる |
| こだわりメガネ | 特殊火力が大きく上がる | 同じ技しか選べなくなる |
| こだわりスカーフ | 素早さが上がる | 同じ技しか選べなくなる |
| たべのこし | 毎ターン少し回復する | 短期決戦では回復量が足りないことがある |
| オボンのみ | HPが減った時に1回大きめに回復する | 使うと無くなる |
| ゴツゴツメット | 接触してきた相手を削る | 非接触技には効かない |
| とつげきチョッキ | 特殊耐久を上げる | 変化技を選べなくなる |
| ひかりのねんど | 壁のターンを伸ばす | 壁を使わないと意味が薄い |
| メンタルハーブ | ちょうはつなどを1回解除しやすい | 1回使うと無くなる |
| あつぞこブーツ | ステルスロックなどの設置技を受けにくい | 火力や耐久は直接上がらない |

「強い持ち物」は、いつでも強いわけではありません。
たとえば、きあいのタスキは設置技でHPが削れると価値が下がります。
こだわり系は火力や素早さを上げますが、技を固定されるため、相手に読まれやすくなります。

## 役割別の選び方

| 役割 | 合いやすい持ち物 | 理由 |
| --- | --- | --- |
| 先発で場を作る | きあいのタスキ、メンタルハーブ、ひかりのねんど | 1回動いて設置や壁を通したい |
| 上から攻撃する | こだわりスカーフ、ブーストエナジー、きあいのタスキ | 素早さか行動保証がほしい |
| 高火力で崩す | いのちのたま、こだわりハチマキ、こだわりメガネ | 相手の受け先に大きな負担をかける |
| 長く受ける | たべのこし、オボンのみ、ゴツゴツメット | 回復や削りでターンを稼ぐ |
| 特殊技を受ける | とつげきチョッキ、たべのこし、オボンのみ | 特殊耐久や継続力を伸ばす |
| トリックルームを使う | メンタルハーブ、オボンのみ、きあいのタスキ | 発動前に止められにくくする |
| トリックルームで攻める | いのちのたま、こだわりハチマキ、くろいてっきゅう | 遅い高火力を通す |

初心者は、1体ごとに「この持ち物がないと何に困るか」を言えるようにします。
言えない時は、火力、素早さ、耐久、行動保証のどれが足りないかに戻ります。

## 強いがデメリットも大きい持ち物

| 持ち物 | 強い使い方 | 注意点 |
| --- | --- | --- |
| こだわり系 | 高火力や高速で同じ技を押しつける | 変化技や別の攻撃技へ切り替えられない |
| いのちのたま | 技を選びながら火力を上げる | 反動で先制技圏内に入りやすい |
| きあいのタスキ | 初手で最低1回の行動を狙う | HP満タンでないと発動しない |
| とつげきチョッキ | 特殊技を受けながら攻撃する | 変化技を使う型とは合わない |
| じゃくてんほけん | 弱点技を受けて一気に火力を上げる | 弱点を受けても倒されると意味がない |
| かえんだま | こんじょう、からげんきなどと合わせる | 毎ターンやけどダメージを受ける |
| どくどくだま | ポイズンヒール、からげんきなどと合わせる | 普通のポケモンには大きな負担 |
| くろいヘドロ | どくタイプが回復できる | どくタイプ以外が持つとダメージになる |
| くろいてっきゅう | 遅くしてトリックルームで動かす | 通常時は後手になりやすい |

持ち物のデメリットは、相手に利用されることがあります。
こだわり系を読まれると、相手は半減や無効のポケモンへ交代しやすくなります。
くろいヘドロやこだわり系は、トリックやすりかえで押し付ける使い方もあります。

## 持っていることが弱点になる技

持ち物は基本的に得ですが、相手の技によっては「持っていること」が狙われます。
特に、ポルターガイスト、はたきおとす、トリック、すりかえは覚えておく価値があります。

| 技・効果 | 何をされるか | 初心者が見ること |
| --- | --- | --- |
| ポルターガイスト | 相手が持ち物を持っている時に当たり、持っていないと失敗する | 持ち物を消費した後なら受け方が変わる |
| はたきおとす | 持ち物を落とされ、持っている相手には威力も上がりやすい | 失うと困る持ち物かを見る |
| トリック | 自分と相手の持ち物を入れ替える | こだわり系を押し付けられると動きが縛られる |
| すりかえ | トリックと同じように持ち物を入れ替える | 受けポケモンがこだわり系を渡されると危険 |
| どろぼう | 攻撃しながら持ち物を奪われることがある | 奪われた後の役割を考える |
| ほしがる | 攻撃しながら持ち物を奪われることがある | 重要な1回きりアイテムを守れるか |
| なげつける | 使った側が自分の持ち物を投げて失う | かえんだま、どくどくだまなどは追加効果狙い |
| アクロバット | 使う側が持ち物を持っていない時に強くなる | 自分が使うなら持ち物を消費する流れを考える |
| ついばむ、むしくい | きのみを食べられることがある | きのみ前提の耐久計算が崩れる |
| やきつくす | きのみなどを失うことがある | きのみ頼みの受け方に注意 |

ここで大事なのは、「持ち物を持つのが悪い」という意味ではありません。
持ち物で得をする代わりに、相手の妨害技で失うリスクもある、という見方です。

## ポルターガイストの考え方

ポルターガイストは、相手が持ち物を持っている時に成立するゴーストタイプの攻撃です。
相手が持ち物を消費した後や、もともと持ち物がない時は失敗します。

| こちらの状態 | 相手のポルターガイスト | 読み方 |
| --- | --- | --- |
| 持ち物をまだ持っている | 当たる条件を満たす | ゴースト弱点なら大きな危険 |
| オボンのみなどを消費した後 | 失敗する可能性がある | 消費後は受け方が変わる |
| はたきおとすで持ち物を失った後 | 失敗する可能性がある | 持ち物を失うことが防御になる場面もある |
| きあいのタスキが残っている | 当たる条件を満たす | タスキで耐えるか、先に動けるかを見る |

ただし、持ち物を失えばいつでも得というわけではありません。
たべのこしを失えば継続力が落ち、あつぞこブーツを失えば設置技が重くなり、こだわりスカーフを失えば先に動けなくなることがあります。

## 持ち物を失うと困る例

| 持ち物 | 失った時に困ること | 例 |
| --- | --- | --- |
| あつぞこブーツ | ステルスロックやまきびしを受ける | 交代回数が多いポケモンほど困る |
| きあいのタスキ | 行動保証がなくなる | 初手で仕事をする予定が崩れる |
| こだわりスカーフ | 素早さで勝てなくなる | 終盤の詰め役が先に倒される |
| たべのこし | 回復しながら受ける計画が崩れる | 受け回しが短くなる |
| ひかりのねんど | 壁ターンが短くなる | 展開構築の猶予が減る |
| しんかのきせき | 耐久が大きく落ちる | 受ける前提が崩れる |

相手に、はたきおとすやトリックが見える時は「落とされても勝ち筋が残るか」を考えます。
持ち物を守るために交代するのか、落とされても仕事を優先するのかを選びます。

## 持ち物がない方が得する場面

持ち物は強いですが、持ち物がない方が得になる場面もあります。

| 場面 | なぜ得か | 注意 |
| --- | --- | --- |
| アクロバットを使う | 持ち物がないと威力が上がる | 最初から持たないか、消費アイテムと合わせる |
| ポルターガイストを受ける | 持ち物がないと失敗する | 持ち物を失うデメリットも大きい |
| はたきおとすを受ける | 落とされる物がなく、威力上昇も受けにくい | そもそも持ち物の恩恵を捨てている |
| かるわざを使う | 持ち物を失った後に素早さを上げる | 消費のタイミングを作る必要がある |

初心者は、基本は持ち物を持たせてよいです。
「持たない方が強い」は、アクロバットやかるわざなど、理由がある時だけ考えます。

## 例

先発でステルスロックを置きたいポケモンなら、きあいのタスキが分かりやすいです。
相手に一撃で倒されても、HP満タンなら耐えて設置できる可能性があります。

ただし、相手が連続技を持っている、天候で削れる、先にステルスロックを置かれる、はたきおとすでタスキを落とされる、などで計画は崩れます。
持ち物は「絶対に仕事をする保証」ではなく、「仕事をしやすくする補助」として見ます。
