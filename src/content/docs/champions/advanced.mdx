## このページで覚えること

- 流行情報は、基本を覚えた後に使う
- 強い名前より、強い理由を見る
- Pokémon Champions でも、まずはタイプ、行動順、役割を見る

## 情報の優先度

新しいゲームや新しいシーズンでは情報が多くなります。初心者は、次の順番で見ます。

| 優先度 | 見る情報 | 理由 |
| --- | --- | --- |
| 高 | 使えるルール、形式 | 使えないポケモンを考えても意味がない |
| 高 | タイプと素早さ | どの環境でも判断に使う |
| 中 | よく見るポケモン | 対策の目安になる |
| 低 | 細かい流行順位 | 最初は振り回されやすい |

## 強い理由を見る

「このポケモンが多い」と聞いたら、名前だけで覚えず理由に分けます。

| 強い理由 | 見ること | 対策の考え方 |
| --- | --- | --- |
| 火力が高い | 受けられる味方がいるか | 半減、耐久、先に倒す |
| 素早い | こちらが先に動けるか | 先制技、素早さ操作 |
| 耐性が良い | 技が通りにくいか | 弱点技、削り、状態異常 |
| 展開が強い | 準備を止められるか | 起点にしない |

## 構築記事やレンタルを使う時

強い構築を使う時は、そのままコピーする前に次の表を埋めます。

| 書くこと | 例 |
| --- | --- |
| 最後に誰で勝つか | 速いアタッカーを残す |
| 不利対面で誰に引くか | でんき技はじめんタイプへ |
| どの相手が苦手か | 受けが硬い相手 |
| 初手で何をするか | 相手の初手を見て攻撃か交代 |

## 例

あるポケモンが流行っている時、その理由が火力なのか、素早さなのか、耐性なのかで対策は変わります。
名前だけで対策するのではなく、「何で負けるか」を表にします。
