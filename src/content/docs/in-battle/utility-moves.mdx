## このページで覚えること

- 攻撃技以外にも、勝ちに近づく技がある
- まきびしやステルスロックは、相手の交代にダメージを与える
- 補助技は「今すぐ倒す技」ではなく「次のターン以降を有利にする技」として見る
- 強さは技単体ではなく、こちらの構築と相手の動きで変わる

## 攻撃しない技とは

対戦では、相手に直接ダメージを与える技だけを押しても勝てないことがあります。
相手が交代する、回復する、積む、こちらより速いなど、単純な撃ち合いにならない場面があるからです。

ゲーム内の分類では、直接ダメージを目的にしない技は主に「変化技」です。
対戦記事や動画では、まとめて「補助技」「絡め手」と呼ばれることもあります。
このページでは、初心者が読みやすいように、攻撃技以外の勝ちに近づく技をまとめて「補助技」と呼びます。
その中でも、相手の場に残って交代時に効果を出すものを「設置技」として覚えます。

| 種類 | 何をする技か | 代表例 | 初心者の見方 |
| --- | --- | --- | --- |
| 変化技 | ゲーム内の技分類。直接ダメージ以外の効果を持つ | ステルスロック、つるぎのまい、ちょうはつ | まずは「殴らない技」と見る |
| 設置技 | 相手の場に効果を置く | ステルスロック、まきびし、どくびし、ねばねばネット | 交代する相手に損をさせる |
| 壁 | 味方が受けるダメージを減らす | リフレクター、ひかりのかべ | 後続を安全に動かす |
| 状態異常 | 相手の動きを弱くする | おにび、でんじは、どくどく、あくび | 火力、素早さ、居座りを制限する |
| 能力変化 | 能力を上げ下げする | つるぎのまい、りゅうのまい、いやなおと | 次の攻撃や受けを強くする |
| 回復・守り | HPやターンを管理する | じこさいせい、まもる、みがわり | 負けにくくして相手を消耗させる |

「通常技じゃない間接技」と感じたものは、まず「今すぐ倒す技ではなく、次のターン以降の得を作る技」と考えると整理しやすいです。

## 設置技は交代に強い

設置技は、置いたターンだけ見るとダメージがありません。
しかし、相手が交代するたびに効果が出るため、長い試合ほど強くなります。

```txt
設置技の価値 =
  相手が交代する回数 × 交代するたびに入る削りや弱体化
```

たとえばステルスロックで等倍の相手が4回出入りすると、合計では最大HPの半分くらいの削りになります。
この削りで、もともと耐えられた攻撃が耐えられなくなることがあります。

<MermaidDiagram title="設置技で相手を削る流れ" caption="相手が交代するほど、後続の攻撃圏内に入りやすくなる。">
{`

  B --> C[出てきた相手が削れる]

</MermaidDiagram>

初心者は、設置技を「交代する相手への間接ダメージ」として見ると分かりやすいです。
相手が何度も交代して受けようとするほど、削りが積み重なります。

## 代表的な設置技

| 技 | 効果 | 強い相手 | 注意 |
| --- | --- | --- | --- |
| ステルスロック | 相手が出てくるたびに、いわタイプ相性に応じたダメージ | ひこう、ほのお、むし、こおりなど、いわ弱点がいる相手 | いわ半減の相手には少なめ |
| まきびし | 地面にいる相手が出てくるたびにダメージ。重ねると増える | 交代を多く使う相手、地面にいる相手が多い構築 | ひこうタイプや浮いている相手には基本効かない |
| どくびし | 地面にいる相手が出てくる時にどく状態にする。重ねるともうどく | 耐久寄り、回復しながら粘る相手 | どく無効の相手には効かず、どくタイプに回収されることがある |
| ねばねばネット | 地面にいる相手が出てくる時に素早さを下げる | 素早さで勝つ相手、こちらの中速アタッカーを通したい時 | 浮いている相手には基本効かない |

### ステルスロック

ステルスロックは、相手が場に出た時にダメージを与えます。
ダメージは「いわタイプの技を受けた時の相性」で変わります。

| 相手のいわ相性 | 入るダメージの目安 |
| --- | --- |
| 4倍弱点 | 最大HPの半分くらい |
| 2倍弱点 | 最大HPの4分の1くらい |
| 等倍 | 最大HPの8分の1くらい |
| 半減 | 最大HPの16分の1くらい |
| 4分の1 | 最大HPの32分の1くらい |

たとえば、ひこうタイプやほのおタイプを何度も出し入れする相手には強く働きます。
一方で、いわを半減する相手ばかりなら、置く価値は少し下がります。

### まきびし

まきびしは、地面にいる相手が場に出た時にダメージを与えます。
1回だけでなく、複数回重ねるとダメージが増えます。

| 重ねた回数 | ダメージの目安 |
| --- | --- |
| 1回 | 最大HPの8分の1 |
| 2回 | 最大HPの6分の1くらい |
| 3回 | 最大HPの4分の1 |

まきびしは、相手が交代で受ける試合ほど強いです。
ただし、ひこうタイプや浮いている相手には基本的に当たらないため、相手の6体を見てから価値を判断します。

### どくびし

どくびしは、ダメージそのものではなく、出てきた相手をどく状態にします。
1回ならどく、2回ならもうどくとして覚えると十分です。

どくびしは、相手が回復しながら粘る時に強いです。
ただし、どくタイプの相手が出ると取り除かれることがあります。
はがねタイプ、どくタイプ、浮いている相手などには効きにくいため、相手の6体を見てから置く価値を考えます。

たとえば、現在の多くの対戦ルールで見るゲンガーのように、地面にいるどくタイプが出ると、どくびしは回収されます。
一方で、ひこうタイプ、ふゆう、ふうせんなどで地面にいないどくタイプは、どくびしを回収しないことがあります。
古い世代や作品では特性が違う例もあるため、「どくタイプなら必ず回収」ではなく「地面にいるどくタイプが回収」と覚えます。

### ねばねばネット

ねばねばネットは、出てきた相手の素早さを下げます。
相手を直接削る技ではありませんが、こちらの少し遅いポケモンが先に動けるようになることがあります。

「上から攻撃できれば勝てる」構築では強いですが、先制技、トリックルーム、浮いている相手には価値が下がることがあります。

## ざっくり強さ順

これは絶対の順位ではありません。
初心者が「どれから覚えるか」を決めるための、普遍的な目安です。

| 目安 | 技やカテゴリ | 強い理由 | 弱くなる時 |
| --- | --- | --- | --- |
| 最優先で覚える | ステルスロック | 多くの試合で腐りにくく、交代に毎回圧をかける | 相手があまり交代しない、いわ半減が多い |
| 強いが相手を選ぶ | まきびし | 交代を多く使う相手を大きく削れる | 浮いている相手が多い、置く余裕がない |
| 構築次第で強い | ねばねばネット | 素早さ関係を変えて、こちらの攻撃を通しやすくする | 相手が浮いている、先制技や素早さ操作で対処される |
| 相手次第で強い | どくびし | 長い試合で回復や受けを崩しやすい | どくタイプ、どく無効、短期決戦 |
| 目的が明確なら強い | 壁、あくび、でんじは、おにび | 後続を安全に出す、相手の行動を制限する | 何につなぐか決めていない |

「強い設置技を全部入れる」より、「置いた後に誰で勝つか」まで決める方が大事です。

## 設置以外でよく見る補助技

設置技以外の補助技は、相手の行動を止める、味方を安全に出す、相手を弱らせるために使います。
強さは「その技を使った後に、こちらが何を得るか」で決まります。

| 技 | 何をするか | 強い使い方 |
| --- | --- | --- |
| ちょうはつ | 相手に攻撃技しか使わせにくくする | 回復、設置、積み、壁を止める |
| アンコール | 相手を直前に使った技へ縛る | 積み技、回復技、まもるを使った相手を起点にする |
| あくび | 次のターンに眠らせる圧をかける | 相手の交代を誘い、設置技の削りと合わせる |
| おにび | やけどで物理火力を落とす | 物理アタッカーを受けやすくする |
| でんじは | まひで素早さを落とす | 速い相手を後続で抜けるようにする |
| リフレクター | 物理ダメージを減らす | 物理相手の前で積みエースを出す |
| ひかりのかべ | 特殊ダメージを減らす | 特殊相手の前で後続を安全に動かす |
| まもる | 相手の技を見る、ターンを進める | どく、やけど、天候、回復のターンを稼ぐ |
| みがわり | 状態異常や読み外しを受けにくくする | 交代や補助技を読んで安全な身代わりを残す |

### 妨害技の強さ順

妨害技は相手によって強さが大きく変わります。
初心者は、次の順で「止めたい行動があるか」を見ます。

| 優先 | 見ること | 例 |
| --- | --- | --- |
| 1 | 相手が回復や設置に頼っているか | ちょうはつで止める |
| 2 | 相手が積み技やまもるを押しそうか | アンコールで縛る |
| 3 | 相手を交代させたいか | あくびで流す |
| 4 | 相手の火力や素早さを下げたいか | おにび、でんじは |
| 5 | 味方を安全に出したいか | リフレクター、ひかりのかべ |

たとえば、相手が回復技を何度も使って受けてくるなら、ちょうはつは攻撃技以上に試合を動かすことがあります。
相手がつるぎのまいを使った直後なら、アンコールで同じ技に縛って、こちらの交代や積みの起点にできることがあります。

## いつ使うか

補助技は、押すタイミングが大事です。
相手に自由なターンを渡すと、その間に倒されたり積まれたりします。

| 使いやすい場面 | 理由 |
| --- | --- |
| 相手が交代しそう | 置くターンを作りやすい |
| こちらがすぐ倒されにくい | 1ターン使っても損しにくい |
| 相手の攻撃を半減できる | 置いた後も動ける |
| 後続のエースを通したい | 壁、ネット、状態異常が勝ち筋につながる |

| 使いにくい場面 | 理由 |
| --- | --- |
| 目の前の相手に倒される | 置くだけで1体失いやすい |
| 相手が積み技を使えそう | 起点にされる危険がある |
| 置いても相手に効きにくい | ターンの損になりやすい |
| こちらの勝ち筋と関係ない | 置いた意味が薄い |

## 相手の設置技への対処

設置技は置かれた後の動きも大事です。
毎回交代していると、出るたびに削られてしまいます。

| 対処 | 何をするか | 覚え方 |
| --- | --- | --- |
| 交代を減らす | 無駄な出し入れをしない | 削りを受ける回数を減らす |
| 早めに攻める | 置いた相手を倒す、起点にする | 置くターンの隙を使う |
| 除去する | こうそくスピン、きりばらいなどで取り除く | 構築に除去役を入れる |
| 無効にする | ひこうタイプ、ふゆう、あつぞこブーツなどを使う | 効かないポケモンを活用する |
| どくびしを回収する | 地面にいるどくタイプを出す | ゲンガーなどが該当することがある |

設置技を置かれたからといって、必ずすぐ除去しなければならないわけではありません。
こちらが交代せずに攻め切れるなら、除去より攻撃が正解になることもあります。

## 変化技に強い特性

補助技や設置技は、タイプ相性だけでなく特性でも止まることがあります。
特に、マジックミラーはステルスロック、まきびし、どくびし、ねばねばネットのような設置技を相手側へ返せることがあるため、設置技を使う前に確認したい特性です。

| 特性・性質 | 止めやすいもの | 注意 |
| --- | --- | --- |
| マジックミラー | ステルスロック、まきびし、どくびし、ねばねばネット、ちょうはつ、でんじは、おにびなど | 使われる前に場にいる時に強い。後から出しても、既に置かれた設置技を消すわけではない |
| おうごんのからだ | 他のポケモンから受ける多くの変化技 | 反射ではなく無効化。攻撃技の追加効果まで全部防ぐわけではない |
| アロマベール | ちょうはつ、アンコール、かなしばり、メロメロなど | 自分や味方の補助技役を止められにくくする |
| ぼうじん | キノコのほうし、ねむりごな、しびれごななどの粉技 | 粉技には強いが、すべての状態異常技を防ぐわけではない |
| ふみん、やるき | ねむり | 眠り技への受け出しを考えやすい |
| じゅうなん | まひ | でんじはへの対策になる |
| めんえき | どく | どくどくやどくびしへの対策になることがある |
| みずのベール | やけど | おにびで物理火力を落とされにくい |

マジックミラー持ちが見えている時に、こちらが初手でステルスロックを押すと、こちら側にステルスロックを置かれる危険があります。
この場合は、先に攻撃する、交代を読む、マジックミラー持ちを削ってから設置する、などの順番を考えます。

一方で、特性で防げる範囲は万能ではありません。
攻撃技で普通に倒されることはありますし、作品やルールによって細かい挙動が変わることもあります。
初心者はまず「設置や状態異常を押す前に、相手に反射・無効化の特性がないか見る」と覚えます。

## 簡単な例

こちらはステルスロックを置きました。
相手には、いわ弱点のひこうタイプがいます。

| ターン | 起きたこと | こちらにとっての意味 |
| --- | --- | --- |
| 1 | こちらがステルスロックを使う | まだ直接ダメージはない |
| 2 | 相手がひこうタイプへ交代する | 出た時に大きく削れる |
| 3 | こちらの通常技で攻撃する | 削り込みで倒しやすくなる |
| 4 | 相手はもう一度交代しにくい | 交代先が削れるため動きが狭くなる |

このように、設置技は「1ターンで得する技」ではなく「相手の行動を少しずつ苦しくする技」です。

## 初心者の使い分け

| やりたいこと | まず考える技 |
| --- | --- |
| 相手の交代に毎回ダメージを入れたい | ステルスロック |
| 受け回す相手を削りたい | まきびし |
| 速い相手を後続で抜きたい | ねばねばネット、でんじは |
| 物理アタッカーを弱くしたい | おにび |
| 後続の積みエースを通したい | リフレクター、ひかりのかべ、あくび |
| 長い試合で相手を消耗させたい | どくどく、どくびし、やどりぎのタネ |

最初は、ステルスロックを基準に考えます。
次に、相手が交代を多く使うならまきびし、素早さで勝ちたいならねばねばネット、長期戦で崩したいならどくびしを考えます。
