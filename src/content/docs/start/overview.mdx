## このページで覚えること

- 最初は「表を見て選ぶ」だけでよい
- 文章、図、表、例、確認テストの順で進める
- 用語や細かい計算は、必要になった時に戻って確認する

## 学習の順番

ポケモンバトルは情報量が多いゲームです。最初から技、特性、持ち物、環境、読み合いを全部覚えようとすると止まります。

最初の目標は、次の4つだけです。

| できるようにすること | 最初の見方 |
| --- | --- |
| 相手に技が効くか | タイプ相性の表を見る |
| 先に動けそうか | 素早さと先制技を見る |
| 攻撃するか交代するか | 倒されそうなら交代候補を見る |
| 6体をどう使うか | 役割表に並べる |

この4つが見えれば、負けた試合も「タイプを見落とした」「交代先がなかった」のように分解できます。

## 非公式用語について

対戦コミュニティでは、ゲーム内に表示されない言葉がよく使われます。たとえば「サイクル」「起点」「一貫」「安定択」などです。

これらは公式名ではありませんが、構築記事や対戦解説を読むためには必要です。このサイトでは、公式用語と区別しながら扱います。

## 例

「相手のこおり技が一貫している」と言われたら、まずは次のように読み替えます。

| 言葉 | 初心者向けの読み替え |
| --- | --- |
| 一貫している | 同じ技を止めるポケモンが少ない |
| 起点にされる | 相手が安全に準備できる |
| 安定択 | 外しても損が小さい選び方 |

## 確認

- 強い構築を真似する前に、表で役割を説明できるようにする
- 非公式用語は、意味が文脈で少し変わることがある
- わからない言葉は [4.1 対戦用語をやさしく読む](/pokedojo/glossary/) に戻る
