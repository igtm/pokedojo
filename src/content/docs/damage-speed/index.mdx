## このページで覚えること

- HPは「あと何回動けるか」を見るための数字
- ダメージには少し幅がある
- ダメージ式は「基本ダメージ × 補正」で読む
- 行動順は、優先度、同じ優先度内の素早さ、場の効果の順に見る

## HPは安全度

HPが多いほど安全、少ないほど危険です。最初は細かい数値より、次の表で見ます。

| 状態 | ざっくりした見方 | 行動の考え方 |
| --- | --- | --- |
| 満タンに近い | 1回は動けそう | 攻撃や準備を考えやすい |
| 半分くらい | 次の攻撃で危ないかも | 相手の弱点技を警戒 |
| 赤い | 先制技でも倒されやすい | 交代や切る判断を考える |

## ダメージは固定ではない

同じ技を同じ相手に使っても、ダメージには少し幅があります。これを乱数と呼ぶことがあります。

| 言葉 | 初心者向けの意味 | 判断 |
| --- | --- | --- |
| 確定1発 | 必ず1回で倒せる | 攻撃しやすい |
| 乱数1発 | 倒せる時と倒せない時がある | 外した時の負け方を見る |
| 確定2発 | 2回当てれば倒せる | 次のターンも動けるか見る |
| 乱数2発 | 2回で倒せるか不安定 | 回復や交代を警戒 |

## ダメージ計算式の読み方

対戦中に式を暗算する必要はありません。覚える目的は、何を上げるとダメージが増えるのか、何を見ると倒せるかを判断しやすくすることです。

```txt
基本ダメージ =
  (((レベル × 2 ÷ 5 + 2) × 技威力 × 攻撃側の能力 ÷ 防御側の能力) ÷ 50) + 2

最終ダメージ =
  基本ダメージ × 補正
```

実際のゲームでは途中で小数の切り捨てや丸めが入ります。ここでは、仕組みを読むために簡単な形で見ます。

| 式の部分 | 何を見るか | 例 |
| --- | --- | --- |
| レベル | 基本は同じルール内なら固定 | ランクバトルはレベル50で考えることが多い |
| 技威力 | 技そのものの強さ | 10まんボルトは威力90 |
| 攻撃側の能力 | 物理ならこうげき、特殊ならとくこう | じしんはこうげき、10まんボルトはとくこう |
| 防御側の能力 | 物理ならぼうぎょ、特殊ならとくぼう | 相手が硬いほどダメージが下がる |
| 補正 | 一致、相性、乱数、急所、天候、やけど、持ち物など | 弱点なら2倍、半減なら1/2倍 |

よく見る補正は次の通りです。

| 補正 | ざっくりした値 | 初心者が見ること |
| --- | --- | --- |
| 乱数 | 0.85倍から1.00倍 | 最小ダメージで倒せるか |
| タイプ一致 | 1.5倍 | 自分と同じタイプの技は強い |
| タイプ相性 | 0倍、1/4倍、1/2倍、1倍、2倍、4倍 | 弱点を突けるか、無効にされないか |
| 急所 | 多くの場合1.5倍 | 狙って前提にするより、起きたら有利くらいに見る |
| 天候 | ほのお、みずなどが増減 | 雨のみず技、晴れのほのお技に注意 |
| やけど | 物理技が下がる | 物理アタッカーは火力が落ちやすい |

## ダメージ計算の簡単な例

次のような条件で考えます。

| 条件 | 数字 |
| --- | --- |
| レベル | 50 |
| 技威力 | 80 |
| 攻撃側の能力 | 120 |
| 防御側の能力 | 100 |
| タイプ一致 | 1.5倍 |
| タイプ相性 | 弱点で2倍 |
| その他の補正 | なし |

```txt
基本ダメージ
= (((50 × 2 ÷ 5 + 2) × 80 × 120 ÷ 100) ÷ 50) + 2
= 約44

最大ダメージ
= 44 × タイプ一致1.5 × 弱点2
= 132

乱数込み
= 132 × 0.85 から 132 × 1.00
= 約112から132
```

相手の残りHPが120なら、112の時は耐えられ、132の時は倒せます。つまり、この例は「乱数1発」です。
相手の残りHPが110なら最小の112でも足りるので「確定1発」、残りHPが140なら1回では倒せません。

## 行動順

多くの通常技は素早さが高い方から動きます。ただし、行動順は素早さだけでは決まりません。
基本は、先に技の優先度を比べ、同じ優先度の中で素早さを比べます。

| 見る順番 | 見るもの | 例 |
| --- | --- | --- |
| 1 | 技の優先度 | 先制技は通常技より先に動く |
| 2 | 素早さ | 同じ優先度なら速い方が先 |
| 3 | 場の効果 | トリックルームなどで同じ優先度内の順番が変わる |

つまり、優先度+1の先制技と優先度0の通常技なら、通常技側の素早さがどれだけ高くても、基本は優先度+1の先制技が先に動きます。
素早さの高さで勝てるのは、基本的に同じ優先度同士の時です。

| こちらの行動 | 相手の行動 | 基本の先手 |
| --- | --- | --- |
| でんこうせっかのような先制技、優先度+1 | 通常技、優先度0 | こちら |
| 通常技、優先度0、素早さ150 | 通常技、優先度0、素早さ100 | こちら |
| 通常技、優先度0、素早さ150 | 先制技、優先度+1 | 相手 |
| 後攻技、優先度が低い技 | 通常技、優先度0 | 相手 |

トリックルームは、優先度そのものを逆転させる効果ではありません。同じ優先度の中で、遅いポケモンが先に動きやすくなる効果として覚えると混乱しにくいです。

終盤では「相手より速いか」だけでなく「先制技で倒されるHPか」も見ます。

## 例

相手の残りHPが55%で、こちらの技が48%から57%入るとします。
倒せる時もありますが、48%なら耐えられます。耐えられた瞬間にこちらが負けるなら、交代や別の削りを考えます。

| 迷う場面 | 見ること | 初心者の答え |
| --- | --- | --- |
| 倒せるか分からない | 最小ダメージで足りるか | 足りないなら危険 |
| 先に動けるか分からない | 優先度と素早さ | 先制技を警戒 |
| 外したら負ける | 外した時の結果 | 安全な行動を探す |
