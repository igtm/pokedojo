## このページで覚えること

- 最初はシングルで、1対1の有利不利を覚えると分かりやすい
- ダブルは2体同時に動くので、考える量が増える
- どちらでも、タイプ相性と行動順は大事

## 形式の違い

| 比べる場所 | シングル | ダブル |
| --- | --- | --- |
| 場に出る数 | 1体ずつ | 2体ずつ |
| 最初に覚えること | 目の前の有利不利 | 2体の組み合わせ |
| よくある行動 | 攻撃、交代 | 集中攻撃、守る、範囲技 |
| 初心者の難しさ | 交代先を選ぶ | 考える対象が多い |

シングルは、まず「目の前の相手に強いか」を見やすい形式です。
ダブルは、相手が2体いるため「どちらを攻撃するか」「隣の味方を守るか」も考えます。

## 最初の練習順

<MermaidDiagram title="初心者向けの練習順" caption="形式より先に、見る順番を固定する。">
{`

  A[シングルで1対1を見る] --> B[タイプ相性を覚える]

| 練習 | できるようになること |
| --- | --- |
| 1対1を見る | 攻撃するか交代するかを選べる |
| タイプを見る | 弱点、半減、無効を避けられる |
| 交代先を見る | 不利な時にすぐ倒されにくくなる |
| 2体を見る | ダブルの集中攻撃や守るを考えられる |

## 共通する考え方

形式が変わっても、最初に見ることは同じです。

| 見ること | シングルでもダブルでも大事な理由 |
| --- | --- |
| タイプ相性 | 技が通るか、受けられるかが決まる |
| 残りHP | 次に倒されるかが分かる |
| 行動順 | 先に倒せるか、先に倒されるかが変わる |
| 交代先 | 不利な時に立て直せる |

## 例

シングルで、こちらのみずタイプが相手のほのおタイプに強いと分かる。
これができると、ダブルでも「こちらの2体のうち、どちらでほのおタイプを攻撃するか」を考えやすくなります。
