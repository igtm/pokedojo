## このページでできること

- タイプ相性をその場で計算する
- 6体のタイプ補完を見る
- 代表例でダメージ幅を確認する
- 構築チェックリストで穴を探す

## どのツールを使うか

| 困っていること | 使うツール | 見る結果 |
| --- | --- | --- |
| 技が効くか分からない | タイプ相性 | 0倍、半減、弱点 |
| 6体の弱点が重なっているか知りたい | タイプ補完 | 一貫しやすい攻撃タイプ |
| 倒せるか確認したい | ダメージ計算 | ダメージ幅、確定数 |
| 6体の役割が足りているか知りたい | 構築チェック | 足りない役割 |
